# Estimaciones-ODS

Este proyecto realiza las estimaciones de los indicadores de los Objetivos de Desarrollo Sustentable (ODS) 3.8.2 de Chile. Tomando como fuente de información directamente las bases de datos disponibles en la web: Encuesta de Presupuestos Familiares (EPF), Índice de Precios al Consumidor (IPC) y Paridad del Poder Adquisitivo (PPA) públicas por el Banco Mundial acorde a la CPI 2017.
